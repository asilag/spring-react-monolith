# Spring React Monolith Template

---

## Overview

This project serves as a foundational template for developing applications using React and Spring Boot.

## Technologies Used

### Backend
- **Spring Web**
- **GraphQL**
- **Spring Data JDBC**
- **PostgreSQL**
- **Lombok**
- **Flyway**
- **Testcontainers**

### Frontend
- **React**
- **Bootstrap**

### Build Tool
- **Maven**

## Key Features

- **Role-Based Security**
- **GraphQL Communication**
- **JDBC Repository**
- **Frontend Maven Plugin**
